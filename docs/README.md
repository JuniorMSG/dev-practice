### 1. 템플릿 사용
`docs/templates/` 폴더의 템플릿을 복사해서 사용하세요.

```bash

```

- ✅ Java와 Kotlin 코드 예제를 함께 제공
- ✅ 실전 활용 예시 포함
- ✅ 주의사항과 함정 설명
