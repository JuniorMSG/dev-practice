# 프로그래머스 문제 풀이

프로그래머스 코딩테스트 연습 문제 풀이를 저장하는 폴더입니다.

## 파일 명명 규칙

```
Programmers{문제번호}.java
```

**예시:**
- `Programmers42576.java` - 프로그래머스 42576번 문제
- `Programmers12345.java` - 프로그래머스 12345번 문제

## 클래스 구조

```java
package src.programmers;

public class Programmers42576 {
    public int[] solution(String[] participant, String[] completion) {
        // 풀이 코드
        return new int[]{};
    }
}
```
