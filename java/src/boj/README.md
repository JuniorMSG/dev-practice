# 백준 (BOJ) 문제 풀이

백준 온라인 저지 문제 풀이를 저장하는 폴더입니다.

## 파일 명명 규칙

```
Boj{문제번호}.java
```

**예시:**
- `Boj1234.java` - 백준 1234번 문제
- `Boj11720.java` - 백준 11720번 문제

## 클래스 구조

```java
package src.boj;

public class Boj1234 {
    public void solve() {
        // 풀이 코드
    }

    public static void main(String[] args) {
        new Boj1234().solve();
    }
}
```
